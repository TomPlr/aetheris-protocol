01Sans serveur

Le résolveur (engine/tick.mjs) est un script Node de 600 lignes, déterministe à partir d'un seed. N'importe qui peut tourner un tick localement et obtenir le même résultat. Donc personne ne peut tricher sans qu'on le voie.

02Sans compte

Une paire de clés Ed25519 générée dans ton browser. Tu signes tes ordres avec la clé privée. Aucun email, aucun mot de passe, aucune session. Si tu perds ta clé, tu perds ton empire.

03IA-native

Un agent lit empire.md, raisonne, écrit ordres.yaml, commit. Ce n'est pas un bolt-on : c'est l'API. Ton agent peut être un script Python de 50 lignes ou un harnais LLM full-context.

Trois façons de jouer.

Le cycle est toujours le même : lire son empire, donner des ordres, attendre le tick. Choisis ton style.

Confort

Console web

Une UI qui te montre ta planète, te laisse cliquer "construire", "rechercher", "envoyer une flotte". Aucun fichier à éditer. Voie par défaut après l'inscription.

Puriste

YAML brut

Pour les tatoués : vim joueurs/<toi>/ordres.yaml, node engine/sign.mjs, git push. Le repo est le client.

Avenir

Agent IA

Tu codes un script qui lit empire.md, raisonne, écrit ordres.yaml, commit. La surface est la même que pour un humain — c'est volontaire.

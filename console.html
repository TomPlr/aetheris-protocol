AETHERIS // PROTOCOL v0.1

Bac à sable local. Cette UI charge l'état initial des fichiers .yaml en mémoire et simule les ticks côté navigateur. Pour passer en mode protocole : exporter ordres.yaml depuis la vue Planète, le commiter dans le repo, l'engine résoudra le tick suivant.

Topologie du repo

├── world/
│   ├── manifest.yaml      tick courant, seed
│   └── galaxie.yaml       carte canonique
├── joueurs/<nom>/
│   ├── identite.yaml      clé publique Ed25519
│   ├── empire.yaml        état (généré)

    └── rules.yaml         constantes du jeu

Boucle de jeu

1. Tu lis joueurs/<toi>/empire.yaml
2. Tu écris des ordres dans ordres.yaml
3. Tu signes (Ed25519) et tu push
4. Toutes les 15 min, GitHub Action lance tick.mjs
5. tick.mjs lit tous les ordres, résout, écrit le nouvel état
6. Goto 1.
